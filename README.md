# Sports-in-US-Metropolitan-Areas
This is a simple analysis project on the sports performance of various US Metropolitan Areas in the different sports leagues (NHL, NFL, NBA, MLB) in the year 2018, particularly finding the correlation between their population and how much their sports teams win or lose games.

## Methodology
**Data Collection**: All data gathered from their relevant wikipedia pages.
**Data Processing**: The project mainly uses Python with Pandas, NumPy, and Re for data pre-processing/cleaning and SciPy for statistics.
