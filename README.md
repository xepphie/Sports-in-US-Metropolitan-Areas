# Sports-in-US-Metropolitan-Areas
This is a simple analysis project on the sports performance of various US/Canada Metropolitan Areas in the different sports leagues (NBA, MLB, NFL, NHL) in the year 2018, particularly finding the correlation between their population and how much their sports teams win or lose games.

## Methodology

### Data Collection 
All data gathered from the relevant wikipedia pages on the 2018 seasons for each sports league.
+ 👩‍👩‍👧‍👦 [US/Canada Population & Sports Data](https://en.wikipedia.org/w/index.php?title=List_of_American_and_Canadian_cities_by_number_of_major_professional_sports_teams&oldid=873225591)
+ 🏀 [2017–18 NBA Season](https://en.wikipedia.org/wiki/2017%E2%80%9318_NBA_season)
+ ⚾ [2018 MLB Season](https://en.wikipedia.org/wiki/2018_Major_League_Baseball_season)
+ 🏈 [2018 NFL Season](https://en.wikipedia.org/wiki/2018_NFL_season)
+ 🏒 [2017-18 NHL Season](https://en.wikipedia.org/wiki/2017%E2%80%9318_NHL_season)

### Data Processing
The project mainly uses *Python* with *Pandas*, *NumPy*, and *re* for data pre-processing/cleaning and *SciPy* for statistics.
